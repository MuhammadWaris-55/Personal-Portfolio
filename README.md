# 🌐 Personal Portfolio Website

Welcome to my personal portfolio website!  
This project showcases my skills, projects, and experience as a Web Developer with a focus on clean UI, modern design, and responsive layouts.

---

## ✨ Features

• Responsive design for all screen sizes <br>
• Clean and modern UI<br>
• Smooth animations and transitions<br>
• Projects section with live demos<br>
• Resume download option<br>
• Contact form with backend integration (Node.js & Express)<br>
• Email sending functionality using SMTP<br>
• Toast notifications using Toastify (instead of alerts)<br>
• Contact section with social links<br>

---

## 🛠️ Technologies Used

• HTML5<br>
• CSS3<br>
• JavaScript<br>
• Node.js<br>
• Express.js<br>
• Nodemailer (SMTP)<br>
• Toastify JS<br>
• Git & GitHub<br>

---

### 📂 Sections Included

• Home – Short introduction and headline<br>
• About – Detailed overview of my skills and journey<br>
• Services – What I offer (Web Dev, UI Design, etc.)<br>
• Portfolio – Selected projects with descriptions<br>
• Contact – Working contact form with backend email service<br>
